# COVID STATS SERVICE

This API provides utils stats about COVID.

## Prerequisites

- MongoDB


## Used Node libraries
- express
- mongoose
- dotenv (just for dev)
- nodemon (just for dev)


## Dev notes
A .env file must be created in root folder with database connection.
```
DATABASE_HOST=mongodb://localhost/covid_stats_db
```
Startup the server
```
npm run devStart
```
